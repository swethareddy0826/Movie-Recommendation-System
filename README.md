# CineMatch — Movie Recommendation System

A Machine Learning-based movie recommendation system built with Python and Streamlit.

## Features
- K-Means Clustering to group similar movies
- Cosine Similarity for intelligent recommendations
- Dynamic filters — Genre, Language, Mood, Director, Actor, Actress
- Preprocessing report with duplicate and missing value detection
- Interactive Plotly visualizations

## Tech Stack
- Python, Streamlit, Pandas, NumPy
- Scikit-learn (KMeans, StandardScaler, Cosine Similarity)
- Plotly
