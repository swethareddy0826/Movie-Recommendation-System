#  CineMatch — Movie Recommendation System

A Machine Learning-based movie recommendation system built with Python and Streamlit.  
Uses K-Means Clustering and Cosine Similarity to recommend movies based on user preferences — no login or watch history needed.

---

##  Tech Stack
Python | Streamlit | Pandas | NumPy | Scikit-learn | Plotly

---

##  Files
- `app.py` — Main application
- `mdr_dataset.csv` — Movie dataset (1000 rows, 640 unique movies)
- `requirements.txt` — Required libraries
